Cómo obtener tu API Key de Google Calendar (gratis):
1. Ve a console.cloud.google.com
2. Crea un proyecto → "APIs y Servicios" → "Habilitar APIs"
3. Busca y activa Google Calendar API
4. Ve a "Credenciales" → "Crear credencial" → Clave de API
5. Copia la clave y pégala abajo. Opcional: restringe la clave a tu dominio.
